# Todo Write

在长任务中，大模型会在一次一次循环中偏离最初的目标，从而导致不理想的结果。

在处理任务时，中途可能会发生错误，从而会将大模型的注意力吸引到处理错误上，为了更好地让大模型在长任务处理过程中保持目的感，我们可以提供一个 todo_write 工具。

让大模型在执行任务时可以自动调用 todo_write 工具，得到需要做的任务清单，后续大模型会根据这些任务清单执行并调用相应的工具，从而得到目标清晰的执行过程。

在 s04 的基础上，添加一个 reminder 机制——通过计数实现，在连续 3 轮循环中没有调用 todo_write 的话，则给大模型一个提醒，让他更新 todos。
